# Clock App ⏰

Welcome to the **Clock App**! This app provides a variety of time-tracking features, including a **Splash Screen**, **Simple Timer**, **Analog Clock**, **Strap Timer (Stopwatch)**, and a **Timer Clock** with rotating time text. The app is built using **Flutter** and **Dart**, providing a smooth, cross-platform experience.

---

## Features 🚀

### 1. **Splash Screen with Logo** 🎬
- The app starts with a **Splash Screen** that displays the app logo, giving a professional and polished first impression.

### 2. **Simple Timer** ⏳
- A **Simple Timer** that changes the **background image** as the timer progresses. This feature adds a dynamic and visually engaging experience.

### 3. **Analog Clock** 🕰️
- An **Analog Clock** displayed in a circular design, just like a traditional clock, showing the time in an elegant format.

### 4. **Strap Timer (Stopwatch)** ⏱️
- A **Strap Timer**, functioning like a **Stopwatch**, allowing you to measure time with start/stop functionality for precision.

### 5. **Timer Clock** 🔁
- A **Timer Clock** where the time is displayed in text format, and the time rotates upside down as it counts down. This adds a unique and fun twist to the standard timer feature.

---

## Technologies Used ⚙️

- **Flutter**: A cross-platform framework for building apps for both Android and iOS with a single codebase.
- **Dart**: The programming language used with Flutter to build the app.

---
## Prerequisites 🛠️

Before running the app, ensure that you have the following installed on your machine:

- Flutter SDK 🚀
- Dart SDK 📦
- A compatible IDE like **Android Studio** or **VS Code** 💻
- Android/iOS simulator or a physical device 📱
---
## Installation 📲

To get the app up and running on your local machine, follow these steps:

1. **Clone the repository**:
    ```bash
    git clone https://github.com/username/clock-app.git
    ```

2. **Navigate to the project folder**:
    ```bash
    cd clock-app
    ```

3. **Install the dependencies**:
    ```bash
    flutter pub get
    ```

4. **Run the app**:
    ```bash
    flutter run
    ```
